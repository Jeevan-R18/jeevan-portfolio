## About

This is a single-page portfolio built to showcase my skills, certifications, and hands-on project work as I work toward a career in cybersecurity. It's built as a static site — no frameworks, no build step, just HTML/CSS/JS.

## Sections

- **About** — background and focus areas (pentesting, network & web security, digital forensics, SOC)
- **Skills** — programming languages, security domains, tools, and networking knowledge
- **Featured Project** — [Security Toolkit Hub](#), a website aggregating common cybersecurity tools with per-tool usage docs, download links, and baseline commands
- **Certifications** — IBM Cybersecurity Fundamentals, Tata Cybersecurity Analyst Job Simulation, glowlogics.in Cybersecurity course
- **Education** — academic timeline
- **Contact** — phone, email, LinkedIn, GitHub

## Tech Stack

- HTML5, CSS3 (custom properties, no framework)
- Vanilla JavaScript (typing animation, respects `prefers-reduced-motion`)
- Fonts: Space Grotesk, Inter, JetBrains Mono (Google Fonts)

## Running Locally

This is a static site — no build step required.

```bash
git clone https://github.com/Jeevan-R18/jeevan-portfolio.git
cd jeevan-portfolio
# open index.html in your browser
```
